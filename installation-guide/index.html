          # Installation guide

          # Introduction

          This document is a guide for installing EasySolder 7 on a Windows system. Depending on your hardware configuration, the installation process can take as little as a few minutes.

          # Installer types

          The following installers are available

          Type Description
          Full Regular installer that also contains all drivers and libraries
          Lightweight Installer that should be used when drivers and libraries are already set up during previous installations. Usually distributed as upgrade to a previous installation

          # Full installer

          Use the regular installer on a new Windows or if EasySolder 7 was not installed before on the PC.

          # Contents

          • EasySolder 7 files
          • Microsoft Visual C++ Redistributable Packages for Visual Studio 2017
          • Microsoft .Net Framework 4.7
          • Rework station DBL driver (Silicon Laboratories CP210x VCP drivers)
          • USB camera driver
          • Sample workspace

          # Lightweight installer

          Use the no-driver, lightweight installer on a PC/tablet where EasySolder 7 or the camera/DBL drivers are already installed.

          # Contents

          • EasySolder 7 files
          • Sample workspace

          # Requirements

          # Hardware

          Category Minimum requirement
          Screen resolution 1024 × 768
          Hard disk space 500 MB
          Memory (RAM) 4GB+
          Ports 2 × USB2+

          # Internet connection

          Internet connection is not needed. All prerequisites are included in the Full installer.

          # Operating system

          The application works on the following 64-bit Windows versions

          • Windows 7 SP1+
          • Windows 8
          • Windows 10

          # Post-installation

          The application should now be visible in the Start menu and Desktop, depending on what options were selected during install.

          After the first application installation a demo workspace is provided and opened at first launch.

          # Launching the application

          The application can be launched from multiple locations described in the sections below.

          # Launch using desktop icon

          This option is available if desktop icon creation was selected at application installation.

          • Go to Desktop
          • Double click on Easy Solder7 application icon

          # Launch using quick launch icon

          This option is available if quick launch icon creation was selected at application installation.

          • Click on Easy Solder7 application icon from toolbar

          # Launch from start menu

          This option is an alternative to open the application when no desktop or quick launch icon was created at installation.

          • Open Start menu
          • Search for Easy Solder7
          • Click on it

          # Rework station info

          If the rework station is connected at launch, it will be recognized by the application and displayed in the bottom bar of the main window with the corresponding firmware id. If the device is not connected, Not connected to device message will be displayed in the bottom bar of the main window.

          # Troubleshooting

          # What to do when the application fails to start after install with the message "This application requires one of the following versions of the .Net Framework"?

          The installer of the application already deploys a .Net Framework. This message means, that during the installation of the application, the .Net Framework encountered an error and was not deployed. Most probably the problem is caused by an unsupported Windows 10 version. The .NET Framework 4.7+ is not supported on:

          • Windows 10 1507
          • Windows 10 1511

          If you're using Windows 10 1507 or 1511 and you want to install the .NET Framework 4.7.2, you first need to upgrade to a later Windows 10 version. Read more here.

          # What to do when the application cannot be uninstalled normally from Control panel

          The application can be uninstalled from the installation path.

          E.g. if the installation path is left the default, then C:\Program Files\Martin\EasySolder 7 select C:\Program Files\Martin\EasySolder 7\unins000.exe
