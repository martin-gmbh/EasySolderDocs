      # Expert 10.6 HV

      Dieses System ist mit einer Hybridheiztechnik ausgestattet. Es ist optimal für mittlere Leiterplatten wie Server-Hauptplatinen.

      Element Beschreibung
      Unterheizung HIF08 (Hybrid)
      Auto-Placement Integriert
      Arm-Kalibrierung Integriert
      Sensoren Sensor 1, Sensor 2, Sensor 3, Sensor 4, Sensor 5, Sensor 6
      Objektivtypen Maxi BGA XL, Maxi BGA, BGA, CSP, Flip-Chip
      Düsentypen BGA, QFN, Micro SMD, QFP

      # Expert 10.6 HXV

      Arm-Kalibrierung Integriert

      # Expert 10.6 IV

      Dieses System enthält eine kompaktere Infrarot-Unterheizung. Es ist für kleine Leiterplatten wie z.B. Handyplatinen konzipiert.

      Element Beschreibung
      Unterheizung IR (Infrarot)
      Auto-Placement Integriert
      Arm-Kalibrierung Integriert
      Sensoren Sensor 1, Sensor 2, Sensor 3, Sensor 4, Sensor 5, Sensor 6
      Objektivtypen Maxi BGA XL, Maxi BGA, BGA, CSP, Flip-Chip
      Düsentypen BGA, QFN, Micro SMD, QFP

      # Expert 04.6 IXH

      Dieses System ist für mittelgroße Leiterplatten konzipiert.

      Element Beschreibung
      Unterheizung IRF02 (Infrarot)
      Auto-Placement Nicht unterstützt
      Arm-Kalibrierung Nicht unterstützt
      Sensoren Sensor 1, Sensor 2
      Objektivtypen Maxi BGA XL, Maxi BGA, BGA, CSP, Flip-Chip
      Düsentypen BGA, QFN, Micro SMD, QFP

      # EXPERT 05.6 IXH

      Dieses System ist mit IR-Heiztechnologie und einem kameragestützten Platzierungsarm (MCP) ausgestattet.

      Element Beschreibung
      Unterheizung IRF02 (Infrarot)
      Placement Manuell (mit Kamera)
      Arm-Kalibrierung Nicht integriert
      Sensoren Sensor 1, Sensor 2
      Düsentypen BGA, QFN, Micro SMD, QFP

      # Sensors

      # Available on devices

      • Sensors available for EXPERT 10.6
        • Sensor 1 is an infrared sensor attached in the device construction. Is controlled by an on/off button.
        • Sensor 2 is integrated on the device construction in the left side of the lamps, is represented by a blue light. The application uses it automatically.
        • Sensors 3, 4, 5 and 6 represented by thermocouples connected to the device sensor ports
      • Sensors available for EXPERT 04.6
        • Sensors 1 and 2 represented by thermocouples connected to the device sensor ports

      # Required to perform operations

      • Sensor 1
        • Used for underheater profiler
      • Sensor 5
        • Used for underheater profiler
      • Sensor 6
        • Used for top-heater profiler
      • The other sensors are optional
